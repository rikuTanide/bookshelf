# bookshelf
積読管理サービス

## 機能

 - ログインしていないユーザー
   - Topページの表示
   - ユーザー一覧を表示
   - ユーザーページを表示
     - 一個でもアイテムがある月が表示される
     - 一月でも表示されている年が表示される
   - サインアップ
 - ログインしているユーザー
   - リストにアイテムを追加
     - 年と月ごと
     - 本・映画・アニメ
   - 本棚から本を削除
   - 本を修正
